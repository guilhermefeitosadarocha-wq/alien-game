NEON SIEGE

Política de Privacidade

1. Dados coletados

O NEON SIEGE coleta apenas os dados mínimos necessários para o funcionamento do jogo: um identificador anônimo gerado automaticamente (sem nome, e-mail ou dados pessoais), pontuação, moedas, upgrades adquiridos e nave selecionada.

2. Como os dados são usados

Os dados são usados exclusivamente para salvar o progresso do jogador entre sessões e exibir o ranking online. Nenhum dado é vendido, compartilhado com terceiros ou usado para publicidade.

3. Armazenamento

Os dados são armazenados localmente no dispositivo (SharedPreferences) e em um servidor Supabase com criptografia em trânsito (HTTPS). O identificador anônimo não permite identificar o jogador pessoalmente.

4. Autenticação

O jogo utiliza autenticação anônima — nenhuma conta, e-mail ou senha é solicitada. Um identificador único é gerado automaticamente e persiste entre sessões.

5. Multiplayer

No modo multiplayer, a posição e estado do jogador são transmitidos em tempo real para o outro jogador na mesma sala via Supabase Realtime. Nenhum dado pessoal é compartilhado — apenas dados de gameplay (posição, tiros, vidas).

6. Permissões

O aplicativo solicita as seguintes permissões: INTERNET (para salvar progresso e multiplayer), ACCESS_NETWORK_STATE (para verificar conectividade) e VIBRATE (para feedback tátil, desativável nas configurações).

7. Crianças

O NEON SIEGE não coleta intencionalmente dados de crianças menores de 13 anos. O jogo não exige cadastro e não solicita informações pessoais.

8. Alterações

Esta política pode ser atualizada ocasionalmente. Alterações significativas serão comunicadas através de uma atualização do aplicativo.
