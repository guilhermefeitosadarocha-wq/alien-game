# 🕹️ Neon Siege

Um jogo de arcade moderno, rápido e viciante feito puramente com **HTML5, CSS3 e JavaScript puro (Vanilla JS)**. Sem frameworks, sem loaders pesados. É só clicar e jogar!

## 🚀 [Clique aqui para jogar agora!]( https://guilhermefeitosadarocha-wq.github.io/alien-game/)

---

## 🎯 O Objetivo
Você precisa sobreviver o maior tempo possível enquanto uma horda de emojis implacáveis persegue você pela tela. Cada segundo sobrevivido conta pontos para o seu recorde!

## 🎮 Como Jogar
* **Movimentação:** Use as teclas `W` `A` `S` `D` (ou as setas do teclado ⬅️ ⬆️ ➡️ ⬇️).
* **Objetivo:** Desvie dos emojis perseguidores.
* **Pontuação:** Sobreviva o máximo que puder e tente bater seu recorde pessoal.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação do jogo e elementos da tela.
* **CSS3:** Estilização moderna, responsividade e animações.
* **JavaScript (JS Puro):** Lógica de perseguição (IA simples), detecção de colisão e sistema de pontuação.
